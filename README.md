## 🛠️ Getting Started & Local Development

### Prerequisites
Make sure you have **Node.js** (v18+ recommended) and **npm** installed on your system.

### 1. Clone the repository
```bash
git clone https://github.com/Wetende/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the local development server
```bash
npm run dev
# OR
npm start
```
This launches the high-speed **Vite** development server. Open your browser and navigate to `http://localhost:5173`. Any edits in `src/` will instantly hot-reload.

## 📦 Building & Production Deployment

### Create an Optimized Production Bundle
```bash
npm run build
```
This compiles, bundles, minifies, and optimizes all assets into the `dist/` folder with `base: '/portfolio/'` configured for GitHub Pages.

### Deploy to GitHub Pages (`gh-pages`)
To deploy the built `dist/` package directly to your GitHub repository's `gh-pages` branch:
```bash
npm run build
cd dist
git init
git branch -M gh-pages
git add .
git commit -m "Deploy production bundle to GitHub Pages"
git push -f git@github.com:Wetende/portfolio.git gh-pages
```
Your live production site will be active at **[https://wetende.github.io/portfolio/](https://wetende.github.io/portfolio/)**.
